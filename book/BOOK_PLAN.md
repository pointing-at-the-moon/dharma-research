## 四、降階重排原則

論文 → 書的轉換,不是重寫,是**降階**。三條原則:

1. **學術腳註轉附錄** — 正文只保留關鍵 CBETA 出處,細節註腳全部後移
2. **跨學科會通段落收緊** — 原論文常有 2,000–4,000 字跨學科段落,降階為 500–1,000 字,保留結構對應核心,刪除鋪墊
3. **補連續性橋樑** — 論文是離散節點,書是連續路徑,節與節之間需要補修行動線的敘事連接

**不做的事**:
- 不重寫論文核心論證(信任已完成的 74 篇)
- 不新增新義理(新義理走新論文路線,不進書)
- 不改動雙軌合流、即非三段式、中道校正等關鍵框架

## 六、Pandoc 編譯工具鏈

**目標輸出**:PDF / EPUB / LaTeX 三種

**技術棧**:
- `pandoc` (核心)
- `xelatex` (PDF 引擎,支援 CJK)
- **字體**:思源宋體(正文)/ 思源黑體(標題)
- `--toc` 自動生成目錄
- `--top-level-division=part` 八部對應 LaTeX \part

**build.sh 骨架**:
```bash
#!/bin/bash
cd book/
pandoc \
  --metadata-file=build/metadata.yaml \
  --toc --toc-depth=2 \
  --top-level-division=part \
  --pdf-engine=xelatex \
  -V CJKmainfont="Noto Serif CJK TC" \
  -V mainfont="Noto Serif CJK TC" \
  -o build/output/指月八部曲.pdf \
  00_總序.md 00_導讀地圖.md \
  parts/01_信/*.md transitions/01_從信到解.md \
  parts/02_解/*.md transitions/02_從解到行.md \
  ... \
  99_終章.md \
  appendix/*.md
```
