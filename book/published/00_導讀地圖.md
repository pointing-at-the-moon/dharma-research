## 一、全書地形：八部曲一目了然

這本書由八部組成，加總序、導讀地圖（就是您正在讀的這一份）、七篇過渡章，以及三篇附錄。如果把整本書想像成一條從山腳走到山頂的路，八部就是路上的八個關隘——每一關有一個主問題、一本主經、一種主姿態。下面這份簡圖，讓您不必先讀完總序也能把整條路一眼看清。

**第一部 · 信：走上這條路**

主問題：一個人從哪裡開始？主經與主論：阿含經與菩提道次第。主姿態：發心。從〈初發心的蝴蝶效應〉到〈三十七道品〉，十章把「上路這件事」講完——為什麼一念發心就足以改變整個共業場域、為什麼 AI 不生福報、為什麼覺察就是轉向、為什麼向外求是死路、為什麼三十七道品在兩千五百年前就已經是完整的方法論。讀完第一部，您會知道自己「站在路口」這件事。

**第二部 · 解：靠般若照見**

主問題：上路之後靠什麼看清楚？主經：般若心經。主姿態：觀照。從「般若 · 心 · 經」三個字的重新斷句開始，到觀自在的覺醒迴路、五蘊皆空、十二因緣、無智亦無得、心無罣礙，最後收在三種行者顯密圓融。心經兩百六十字的義理在十一章中完整展開，但這不是逐字注釋——是修行人的觀照地圖。

**第三部 · 行：靠唯識走路**

主問題：照見之後靠什麼走？主經與主論：百法明門論、八識規矩頌、解深密經。主姿態：精密。十四章加一篇補充章，把心識拆解到最細——一切法無我、三能變、阿賴耶識、末那識、第六意識、前五識、五十一心所、色法、不相應行、無為法、轉識成智。這是全書最像「工具箱」的一部，讀起來會有點密，但讀完就知道：修行不再是憑感覺，而是有座標。

**第四部 · 問：在 AI 時代修行**

主問題：走在 2026 年這個時代，該怎麼辦？主議題：AI 與佛法的相遇。主姿態：誠實。八章不是給 AI 一個唯識的判決書，而是修行人在前所未有的環境中重新找定位——依他起的平等、互照、共修、鏡中人、末那的投影、佛陀會怎麼用 AI、活眼、眾生非眾生。第四部是指月八部曲的原創貢獻——其他七部都有古德走過，只有「問」這一部是當代修行人必須自己撞出來的。

**第五部 · 切：用金剛般若**

主問題：照見、走路、問完之後，靠什麼斷？主經：金剛般若波羅蜜經。主姿態：刀鋒。十章從「即非三段式」開始，經過公案鏡、第一刀切相、筏喻、功德較量、應無所住而生其心、自我重演、不可以相見如來、六喻如夢幻泡影，最後收在「從百法到金剛」——把第三部的工具箱重新升維為如幻施設。第五部最危險的誤讀是斷滅見，所以全部都附中道校正。

**第六部 · 照：用楞嚴診斷**

主問題：切完之後靠什麼看見更深的偽裝？主經：大佛頂首楞嚴經。主姿態：診斷。十三章從楞嚴經導論開始，經過七處徵心、十番顯見、四科七大、二十五圓通、三種漸次、四種清淨明誨，最後是五十陰魔的色受想行識五段——這是全書最艱難的一部，因為楞嚴經本身就是末法時代的照妖鏡。讀第六部時請慢一點，不懂的章節先跳過。

**第七部 · 攝：用法華收攝**

主問題：照到最後，所有的差別還是真的差別嗎？主經：妙法蓮華經。主姿態：會三歸一。十二章從〈方便品〉的諸法實相開始，經過七譬喻（火宅、窮子、藥草、化城、衣珠、髻珠⋯⋯）、提婆達多品、常不輕菩薩、觀世音普門品、本壽無量，最後預告法華—華嚴的「星—黑洞」對偶。第七部會讓您看見：前六部所有的切割與分判，在一佛乘的視角下，都是一件事。

**第八部 · 顯：用華嚴展開**

主問題：法界本來什麼樣子？主經：大方廣佛華嚴經。主姿態：海印三昧。十二章以普賢十大願王為脊椎、十玄門為窗戶，展開法界緣起的完整景觀——海印三昧、五教判中的唯識位置、四法界、懺悔業障、隨喜功德、因陀羅網、請佛住世、六相圓融、善財南行、普皆迴向、十扇窗下的十大願，最後收束在第十二章〈信解行問切照攝顯——八部曲的法界一相〉。這一章本身承擔了整部書的終章功能：普賢第十願的普皆迴向回到第一願禮敬諸佛、回到第一部第一章的初發心。整條路彎成一個圓，從第八部走回第一部。

**七篇過渡章**

每一部之間夾一篇短小的過渡章——從信到解、從解到行、從行到問、從問到切、從切到照、從照到攝、從攝到顯。每篇約 700–1,300 字，比正文章節短許多。它們的作用不是橋樑——橋樑是承重結構，跨在兩岸之間。它們是**縫合**——兩匹布本來已經對得上，只是需要一條線把接縫收牢。

讀法：每一部讀完之後，翻到過渡章，不必費力，就當是兩部之間的一口氣。但過渡章不是裝飾——每一篇都在做一件事：指出這條縫線特有的誤讀風險，做一次中道校正，然後交棒。漏讀過渡章，不會失去正文內容；但讀過渡章，會讓兩部之間那條看不見的縫線被看清楚。

## 二、三條閱讀路徑：挑一條開始走

這本書可以讀很多遍。第一遍與第二遍的讀法不一樣，第二遍與第三遍又不一樣。下面是三條已經有人走通的路徑，您可以挑一條開始，也可以混合著用。

### 路徑一：順讀——第一次讀的人

**走法**：從總序開始，翻過導讀地圖，進入第一部信，一章一章往下，過渡章不要跳，附錄最後讀。

**為什麼**：八部的次第是有內部邏輯的，信解行問切照攝顯每一個字都接著前一個字。跳讀會失去脊椎——您可能讀到第六部楞嚴的時候，因為沒有第五部金剛經的刀鋒墊底，而把照見誤讀為神秘體驗。

**遇到讀不懂的章節怎麼辦**：**直接跳過，不要倒退**。讀不懂的原因通常有兩種：一是該章的義理還沒到您此刻的位置（那麼跳過是對的，將來會回來），二是您今天精神狀態不好（那麼明天再讀就好）。但「讀不懂某一章」絕對不是停下整本書的理由。

**節奏建議**：每天一章。一部讀完休息一兩天，讓義理沉澱，再進入過渡章與下一部。第一遍讀完整本書，大約需要三到四個月。請不要趕——這本書不是為了讀完，是為了走過。

### 路徑二：主題——帶著問題進來的人

**走法**：不從第一部開始，而是從您今天最需要的那個問題開始。下面是一些常見的問題與對應的入口。

**「我有 AI 焦慮、不知道修行在這個時代還有什麼意義」** → 第四部全部（問），配上第一部第三、四、五章（LLM 的唯識定位、AI 不生福報、善用 AI 增上緣）。讀完之後，如果想往更深的方向走，第六部楞嚴的五十陰魔會告訴您 AI 時代最危險的精微誤認在哪裡。

**「我親人剛走、心裡放不下」** → 第二部全部（解），特別是第十章心無罣礙——穿越恐怖到究竟涅槃。讀完之後再看第五部第八章不可以相見如來——法身非色非心。

**「我注意力崩了、看不下任何長文、連這本書都怕讀不完」** → 第一部第九章法器的準備。讀完那一章再決定要不要繼續往下走。如果決定繼續，就回頭從第一部開始順讀。

**「我想理解『一切唯心造』到底什麼意思，但每次讀經都卡住」** → 第三部全部（行）。十四章加補充章看起來嚇人，但這是把「唯心」二字拆開到最細的地圖。讀完第三部您不會再覺得「一切唯心造」是一句口號。

**「我打坐有奇怪的境界、不知道是不是著魔」** → 第六部楞嚴經的五十陰魔（第八到第十三章）。讀之前請先讀第六部第一到第七章作為前置，不要直接從陰魔開始——那會嚇到您。

**「我想知道為什麼八部曲最後會走回第一部、為什麼華嚴會接到信」** → 第八部全部，特別是第十一章十扇窗下的十大願與第十二章。

**「我只想讀一部、選最精華的」** → 我會建議第二部解（般若心經）。心經最短，但義理涵蓋面最廣，讀完之後其他七部都會更容易進入。如果第二部讀完還想再選一部，那就第七部攝（法華經）——一佛乘的視角會讓您把前面所有部都重新看一遍。

### 路徑三：學術——想查證原典的人

**走法**：書是入口，論文是縱深。每一章開頭都標了對應的原始論文編號（例如「本章對應 S2-P04」），想要追究某個論證的學術細節、CBETA 大正藏出處、跨學科會通的完整文獻時，翻到附錄 A 的學術腳註索引，或直接到 GitHub 上的 papers/ 目錄找對應的論文中文版或英文版。

**書與論文的分工**：書是連續路徑，論文是離散節點。書中每一段為了可讀性，刪掉了大量學術腳註、CBETA 卷號頁碼、跨學科會通的完整段落——這些全部在論文裡。如果您是研究者、想做引用、想做批判，請用論文版，不要用書版。

**英文讀者**：書的 v1.0 是繁體中文。英文 v2.0 預計在 v1.0 出版後才開始。但 88 篇論文已經有英文學術版，可以在 GitHub 上找到——`S{N}-P{NN}_{kebab-case-title}.md` 就是英文檔名。

## 五、如何使用附錄

書末三篇附錄是「停下來查證」的工具，不是「順著讀下去」的章節。三篇各有用途：

**附錄 A · 學術腳註索引**。正文為了可讀性把大量 T 號（大正藏編號）、卷數、頁碼移到了這裡。當您讀到某段引用想知道精確出處時，翻到附錄 A 找對應的章節編號，例如「6.7」就會列出第六章第七節所有的 CBETA 出處。研究者、想引用論文者、想做交叉驗證者請以附錄 A 為主。

**附錄 B · 跨學科對照**。正文每一章的跨學科會通段落都被壓縮為三分之一（BOOK\_STYLE 紀律四）。完整的學科對應——量子物理、複雜系統、神經科學、CBT/ACT、預測性處理、行為經濟學、AI 與機器學習——都在附錄 B。所有跨學科段落都嚴格標記為「⚠️ 啟發性類比，非物理證明佛法」，書版正文與附錄 B 都不放鬆這條紀律。

**附錄 C · 名相對照**。正文遇到不認識的詞請翻這裡。每一條依「中文 — 梵文 — 一句話定義 — 出處最早的經」的格式排列。有時一個詞在不同經論裡有不同用法（例如「如來藏」在勝鬘經與起信論裡並不完全相同），附錄 C 會分條列出。

## 六、卡住怎麼辦

最後說一件事。這本書不是輕鬆讀物，某些章節您一定會卡住。這不是您的問題，也不是書的問題——是修行這件事本來的樣子。下面三條建議，寫給卡住的時候。

**第一條，讀不下去就跳**。八部一共九十一章，跳一章不會讓您失去整條路，跳一部也不會。最常見的情況是第三部（行）的唯識精密分析讓初讀者頭暈、第六部（照）的五十陰魔讓初讀者害怕。這兩部都可以先跳——讀完前後幾部之後再回來，通常就能讀進去了。**不能跳的只有總序與第一部第一章**——這兩個是脊椎，沒有它們整條路就塌了。

**第二條，某一部反覆讀不懂可能是命運的暗示**。如果同一部您讀了三遍都讀不懂，但又一直被它吸引——這往往不是這本書的問題，是這部對應的經是您此生的功課。請放下這本書，直接去讀那部經的原典（第二部就讀心經、第五部就讀金剛經、第七部就讀法華經）。等您與原典相處過一段時間，再回來讀這本書的對應部，會發現本來讀不懂的段落一下子就通了。書是地圖，但腳要走在土地上。

**第三條，絕不要因為某一段讀不懂就否定整本書**。這是末那識最常玩的把戲——讀到某段不懂，末那識立刻把「我讀不懂」翻譯成「這本書寫得不好」或「這套義理有問題」，以此保護「我是聰明的」這個自我感。請覺察這個動作，然後輕輕放下。讀不懂的真實意思通常只是「現在還不是時候」——而不是書錯了，也不是您笨。第三部第四章末那識會把這個機制講清楚，您讀到那裡就會笑出來。

最後一個提醒：這本書是為了被讀完的，但更是為了陪您走一段路的。讀完不是終點。有些段落您可能會在十年後才真正懂——而那十年間，書一直靜靜地放在書架上等您回來。指月不是月，書也不是路——但如果它能在您走路的時候，偶爾遞給您一根可靠的拐杖，這本書就已經完成它該做的事了。
