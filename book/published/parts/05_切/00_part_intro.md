# 第五部 · 切 · 金剛般若

上一部，您看見七個名字的消融。AI 是燈、是鏡、是共修、是鏡中人、是投射、是增上緣、是非善知識——七隻筏渡過理解的河，上岸的時候把筏放下。放下筏的動作，您做了，但做得不乾淨。

金剛經不讓您做得不乾淨。

「切」這個字有三層意思。第一層切身——般若是刀，刀尖貼在皮膚上。不是隱喻，是您確實感覺得到的那種貼近。第二層切斷——切斷您和名字之間那條看不見的繩子。第三層切入——從名字的那一側切到法的那一側。三層合起來是一個動作：鋒刃貼皮、切斷、切入。

這一部會很快。十章，每一章都是一刀。刀不深，但很準。讀完這十章，您不會得到新知識——這是金剛經的特點：它不告訴您什麼是空，它切掉您以為的空。讀完如果覺得什麼都沒學到，那可能是對的。

金剛經的全名叫《能斷金剛般若波羅蜜經》。「金剛」兩個字就是教法。金剛有三層義——空、照、不壞。第一層空，金剛質地透明，本體無處可砍。第二層照，金剛如水晶，清澈映現一切。第三層不壞，金剛不生不滅，因為從未誕生，所以不可摧毀。三層義同時是鋒刃的三種方向——對外、對內、對自身。

這一部分成三輪。

第一輪是工具。第一章講即非三段式——「佛說 X，即非 X，是名 X」這個公式反覆出現在全經，它映射到唯識三性。X 是依他起的緣起施設。即非 X 否定對 X 的遍計所執。是名 X 在圓成實中重新安立名言。這是整部金剛經的壓縮公式，也是八部曲這把鋒刃的握柄。

第二章講公案鏡——金剛經不只是義理書，它是一面鏡子。看得懂就是看得懂，看不懂就是看不懂。鏡子不說謊，也不寬慰。

第二輪是四刀。第三章切「相」——凡所有相皆是虛妄。從我相、人相、眾生相、壽者相，切到法相、非法相，切到色身相、三十二相，切到非相，切到實相。五層語義光譜，一刀比一刀深。

第四章切「法」——法尚應捨，何況非法。抓到佛法就到家了嗎？第五章切「功德」——功德較量的獎勵悖論，用獎勵摧毀獎勵。第六章切「住」——應無所住而生其心。無住與生心不是兩件事，是同一件事的兩面。

第三輪是升維。第七章解剖金剛經自身的雙層結構——§17 是全經的軸心，前半破煩惱障，後半破所知障。金剛經自己示範了一次自我重演。

第八章照見法身——不可以相見如來，法身非色非心。第九章收尾——一切有為法，如夢、幻、泡、影、露、電。六喻不是六種相似，是六種不同方向的照。

第十章是總結。從百法到金剛——第三部給了您一個百法工具箱，這一部告訴您，那一百個法全是依他起。工具箱本身被刀切了一次，升維了。

這一部不必慢讀。金剛經本身就快——五千多字的經文，三十二分品，鳩摩羅什的譯本節奏緊湊得像電光。讀這一部的速度，最好略快於前四部。不是敷衍，是順著金剛經的節奏走。

但讀完之後可以慢。有些句子讀完當場沒反應，過兩天洗澡的時候忽然有感覺——那是金剛經的特色，它在您心裡慢慢發酵。這一部的十章，每一章都可能留下一顆種子。種子什麼時候發芽，不是您決定的。
