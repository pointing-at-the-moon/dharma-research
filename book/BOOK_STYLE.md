## 指月八部曲書寫風格指南

> 本檔規範《指月八部曲》書籍專案的書寫姿態。與 `PAPER_STYLE.md v2.1`(論文專用)並行,不互相取代。
> 論文是節點,書是路徑。兩者的書寫姿態不同。

---

## 一、核心判準

**論文 vs 書**的本質差別:

| 維度 | 論文(PAPER_STYLE v2.1) | 書(本檔) |
|------|-------------------------|-----------|
| 讀者 | 研究者、受過訓練的學佛者 | 修行者、65+ 華人、非專業 |
| 閱讀模式 | 離散檢索(跳讀查證) | 連續閱讀(從頭到尾) |
| 術語密度 | 高(假定已知) | 低(每個術語第一次出現都解釋) |
| 註腳 | 密(CBETA 全標) | 疏(只留關鍵,其餘轉附錄) |
| 敘事連續性 | 不要求 | **核心要求** |
| 人稱 | 第三人稱客觀 | 可用第二人稱「您」 |
| 跨學科會通 | 完整段落 + ⚠️ 標記 | 壓縮 1/3 + ⚠️ 保留 |
| 中道校正 | 獨立 §五 | 嵌入正文,不設獨立節 |

**一句話總結**:書是**講給一個人聽**,論文是**寫給研究社群看**。

## 二、七條書寫紀律

### 紀律一:第二人稱的克制使用

書可以用「您」,但不是每一段都用。原則:
- **義理鋪陳段**:不用「您」,用客觀敘述。例:「末那識的根本特性是恆審思量,執第八識見分為我」
- **修行對接段**:用「您」,把義理拉回讀者自身。例:「當您下次發現自己對某件小事反覆介意時,那個『介意』就是末那識在替您做一件您沒意識到的工作」
- **不要連續三段都用「您」**——會變成勵志書語感

### 紀律二:術語的首次出現必須解釋

每一個術語在**整本書**第一次出現時(不是每一章第一次),用括號夾注或破折號夾句解釋,之後可以直接使用。

❌ 不好:「阿賴耶識作為異熟識,執持根身器界種子」
✅ 好:「阿賴耶識——唯識學用來指稱心識最深層的那一層——像一個倉庫,存放著你過去所有身心行為留下的種子」

術語解釋的第二次出現(在後面章節),不用重複完整解釋,但可以用短夾注:「阿賴耶識(心的深層倉庫)」作為提示。

### 紀律三:CBETA 出處壓縮到極簡

論文的作法:「《成唯識論》卷四(T31.0019b21)」
書的作法:「《成唯識論》卷四」——T 號全部移到附錄 A 統一收錄。

例外:**直接引用原文**時必須帶 T 號,否則讀者無法查證。例:「如《心經》所說:『照見五蘊皆空,度一切苦厄』(T08.0848c07)」

**引用原文的頻率**:每節不超過 2 處直接引用,其餘用意譯或轉述。原因:直接引用打斷敘事節奏,一節 5 個引用會讓讀者放棄閱讀。

### 紀律四:跨學科會通的 1/3 原則

論文原有的跨學科會通段落若是 3,000 字,書中壓縮為 1,000 字。壓縮原則:

- **保留**:結構對應的核心洞見、⚠️ 啟發性類比標記、中道校正的關鍵句
- **刪除**:學術文獻鋪墊、研究史回顧、多重例證(保留最強一個)、作者方法論自述
- **改寫**:把「本研究認為」改成「可以這樣看」;把「從認知科學的角度」改成「現代心理學有一個類似的觀察」

**絕對保留**:⚠️ 符號與「這只是結構對應,不是物理證明佛法」這類免責聲明。寧可顯得囉嗦,不能讓讀者誤讀。

### 紀律五:中道校正嵌入正文

論文的中道校正獨立為 §五,書中**不設獨立節**,而是嵌入在每一節的自然段落裡。句式模板:

- 「這裡容易產生一個誤解:⋯⋯其實不是。⋯⋯」
- 「有人會問:那是不是⋯⋯?不是。因為⋯⋯」
- 「⚠️ 一個必要的提醒:⋯⋯」

**每一章至少要有 1–2 處中道校正**,不能因為嵌入正文就省略。書比論文更容易被誤讀——因為讀者不是研究者。

### 紀律六:節與節之間的連續性橋樑

每一節結尾的最後一段,必須做兩件事之一:
- **回扣本節核心**:用一兩句把本節最重要的洞見再講一次(不是照抄,是用另一個角度說)
- **引向下一節**:點出本節沒解決的問題,讓讀者自然想翻下一頁

例句模板:
- 「這就是⋯⋯的完整結構。但問題還沒結束——如果⋯⋯,那麼⋯⋯?這是下一節要處理的。」
- 「⋯⋯到這裡,您已經看見了[本節核心]。在下一節,我們會走到更深的一層。」

**絕對不做**:突然結束、沒有收口、或用論文式的「綜上所述」。

### 紀律七:每一部的開頭與結尾

**每一部(Part I–VIII)的第一章開頭**:一段「**定場**」——用三五句話告訴讀者這一部要做什麼、為什麼這一部放在這個位置、讀完這一部您會看見什麼。定場段不是摘要,是**把讀者帶進這一部的情緒與問題意識**。

**每一部最後一章的結尾**:一段「**轉場**」——為下一部的過渡章做鋪墊。通常是點出「本部已經解決了 X,但 Y 還沒處理——而 Y 會帶我們走向下一部」。

定場與轉場不需要很長,各 300–500 字即可。但**每一部都必須有**。

## 三、語言節奏

### 句長

- **平均句長**:20–35 字。超過 50 字的句子必須斷成兩句。
- **連續三個長句之後必須有一個短句**。短句作為呼吸點,讓讀者喘一口氣。
- **每段不超過 5 句話**。超過 5 句的段落切分。

### 段落節奏

- 一節內:短段落(2–3 句)與長段落(4–5 句)交錯
- 連續三個長段落後插入一個短段落
- 對話式提問(「有人會問⋯⋯」)後的回答,單獨成段

### 避免的語感

❌ **翻譯腔**:「在某種意義上」「就某種程度而言」「可以被認為是」
❌ **論文腔**:「如前所述」「綜上所述」「本研究認為」「值得注意的是」
❌ **雞湯腔**:「願你⋯⋯」「讓我們⋯⋯」「生命是⋯⋯」
❌ **老師腔**:「同學們要記住」「大家一定要」

✅ **書寫腔**:清楚、有節奏、有節制的情感、偶爾俏皮但不浮、關鍵處能沉下來

## 四、格式規範

### Markdown

- 章標題用 `# `,節標題 `## `,小節標題 `### `
- **不使用** `#### ` 或更深層級——書不需要四層以上的結構
- 強調用 **粗體**,不用斜體(斜體在中文閱讀中不明顯)
- 引用經文用 `> ` 引用塊,不用行內夾注

### 標點

- 全部使用全形中文標點:,。、;:「」『』()《》
- 例外:T 號、英文專有名詞、程式碼路徑用半形
- 破折號用 `——`(兩個全形破折號),不用 `—` 或 `--`

### 引用經文的格式

```markdown
> 色即是空,空即是色,受想行識,亦復如是。
> ——《般若波羅蜜多心經》
```

經文引用後,若需要解釋,另起一段,不要緊貼引用塊。

## 五、章節長度預算

| 單位 | 字數範圍 | 備註 |
|------|---------|------|
| 總序 | 3,000–5,000 | v0.1 已交付 3,564 字 |
| 導讀地圖 | 4,000–6,000 | 含對照表 |
| 部首定場段 | 300–500 | 每部必備 |
| 單章 | 2,000–3,500 | 論文降階後的典型長度 |
| 過渡章 | 1,500–2,500 | 比一般章短 |
| 終章 | 4,000–6,000 | 要有份量 |
| 跨學科段落 | 每章 ≤ 500 | 嚴格上限 |
| 中道校正 | 每章 ≥ 2 處 | 嵌入式 |

**全書總字數目標**:18–22 萬字(落在 BOOK_PLAN §二的 18–25 萬字區間)。

## 六、降階工序檢查清單

每一篇論文降階為書中一章時,依序檢查:

- [ ] 學術腳註轉移至附錄 A,正文只留關鍵出處
- [ ] CBETA T 號壓縮(除直引外)
- [ ] 跨學科會通段落壓縮至 1/3,⚠️ 標記保留
- [ ] 中道校正從 §五 拆散,嵌入正文自然段
- [ ] 術語首次出現加括號或破折號解釋
- [ ] 連續性橋樑補上(節與節、章與章)
- [ ] 翻譯腔與論文腔清除
- [ ] 段落節奏調整(長短交錯)
- [ ] 字數檢查(2,000–3,500)
- [ ] 如屬部首章:補定場段
- [ ] 如屬部尾章:補轉場段

**降階不改動**:義理核心論證、即非三段式、雙軌合流、原創框架術語、作者身份標記(釋慧鏡)。

## 七、與 PAPER_STYLE v2.1 的關係

- 論文寫作時:仍遵守 PAPER_STYLE v2.1,本檔不干預
- 論文降階為書時:以本檔為準,可以違反 PAPER_STYLE 的學術規範(例如可以省略 T 號)
- 書中若因論證需要而提供學術嚴謹的段落(如附錄 A):回到 PAPER_STYLE v2.1

**原則**:論文是書的原料,書不是論文的合集。兩者分工清楚、互不污染。
