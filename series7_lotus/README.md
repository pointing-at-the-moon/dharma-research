# Series 7：攝 — 妙法蓮華經
**Gather (攝) — The Lotus Sūtra: Drawing All Vehicles Back into One**

S1–S3 建構了唯識地圖，S4 以質疑拆解盲信，S5（金剛經）把一切執著切掉，S6（楞嚴經）提供驗證真偽的照妖鏡。S7（法華經）的工作不是建立新的執著，而是**回收**——被切掉的、被驗出錯誤的、走歪了的、走對了的，全部攝入一佛乘。沒有 S5 的切，S7 的攝就變成鄉愿；有了 S6 的照，S7 的攝才能辨別方便與究竟。

本系列以三個學術命題貫穿：**命題一**——法華「方便—實相」結構 = 唯識「遍計—圓成」的三層語域轉換（世親虔敬語域→吉藏本體語域→窺基分析語域）；**命題二**——法華與華嚴構成「攝—顯」對偶（恆星向外輻射 vs. 黑洞向內坍縮）；**命題三**——法華七喻為阿含種子的戲劇化完成。

### S7-P01. 諸法實相·十如是——法華經的義理基石
**The True Mark of All Dharmas and the Ten Suchnesses: The Doctrinal Keystone of the Lotus Sūtra**

追溯「唯佛與佛乃能究盡諸法實相」在法華詮釋史上的三層語域轉換：世親《法華論》（T1519）以「法身平等」為虔敬語域——全論完全不使用三性術語；吉藏《法華義疏》（T1721）引入「如來藏法身之體」的本體語域；窺基《法華玄贊》（T1723）完成「依他起即菩提、圓成實即涅槃合成一乘」的分析語域。三者以依他起性為共同樞紐，非互斥而為同一實相的三種語域表達。以正法華經對勘、智顗三轉讀天台對照、護法依他起樞紐段為輔助論據。為 S7 全系列奠定文本基石，並預告法華—華嚴攝—顯對偶命題。
