## 系列簡介

以如來藏語言建立信心，從現代人的痛點切入。十篇論文構成一個完整的「從零到方法」的修行入門：

1. 發心的力量（#1-2）
2. AI 時代的因果觀（#3-5）
3. 覺察、因果、向內（#6-8）
4. 準備與方法（#9-10）
