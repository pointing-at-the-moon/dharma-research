## 6. Lessons Learned

| 教訓 | 規則 |
|------|------|
| 複製檔案後再修改 = 副本過時 | 所有修改完成後才做最終複製，或修改後重新同步 |
| 中文論文禁用隨意英文 | 年長讀者看不懂，學術括號術語可保留但正文不混英文 |
| 審計要檢查腳注交叉引用 | 改了標題要同步更新所有引用該標題的腳注 |
| 不能直接說讀者「有漏」| 用問題引導讀者自我發現，不貼標籤 |
| 善惡不抵 vs 轉依不矛盾 | 但需要明確標示層次差異，否則讀者會困惑 |
