# Series 2：解 — 般若波羅蜜多：照見實相

**Prajñāpāramitā — Seeing Reality As It Is**

## 系列簡介

以般若智慧照見五蘊皆空，從觀自在菩薩的因地修行展開，逐句導讀《般若波羅蜜多心經》。從「般若·心·經」的重新斷句開始，經觀自在、行深、照見五蘊皆空，到色即是空、十二因緣、不生不滅，經「無智亦無得」、穿越恐怖到究竟涅槃，最終以三種行者、顯密圓融收束。
