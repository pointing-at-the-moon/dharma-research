# Series 2：解 — 般若波羅蜜多：照見實相

**PrajnaParamita — Seeing Reality As It Is**

Papers #11–20 · YouTube EP08–16

---

## 系列簡介

以心經五義架構為主線，展開般若智慧的完整圖景。來源：《般若波羅蜜心經導讀——經論校注版》。

從「般若·心·經」的重新斷句開始，經觀自在、行深、照見五蘊皆空，到色即是空、十二因緣、不生不滅，最終抵達「無智亦無得」——連般若本身也不可執。
