# Series 6：照 — 大佛頂首楞嚴經
**Illumination (照) — The Śūraṅgama Sūtra: The Practitioner's Diagnostic Mirror**

S2（心經）以般若照見五蘊皆空，S5（金剛經）以即非之刃切穿一切法的實有性，S6（楞嚴經）將「照」從般若智慧的洞見部署為修行的完整診斷系統。楞嚴經十卷以三同心圓結構展開：內圈驗心（七處徵心、十番顯見）、中圈修法（二十五圓通、四種清淨明誨）、外圈辨魔（五十陰魔）。從「一向多聞未全道力」到「一切事究竟堅固」，楞嚴經回答 S5 結尾留下的問題：那個在用百法地圖、在揮金剛刃的，是誰？
