## 摘要草案

本文重新發現並分析窺基《金剛般若波羅蜜經讚述》（T1702）中對金剛經「即非 X，是名 X」公式的三性映射——「說→遍計所執，即非→依他起，是名→圓成實」——並提出一個替代映射：「X→依他起，即非→否定遍計，是名→圓成實」。兩種映射的差異在於遍計所執的定位：窺基從聽者的角度，本文從對象的角度。本文論證兩種映射互補而非矛盾，共同揭示即非結構作為唯識三性壓縮表達的完整義理機制。文中並以無著（T1510b）、世親（T1511/T1512）的注疏為進程脈絡，呈現從「結構分析而未用三性術語」到「明確三性映射」的學術演進。

## 一、引言

### 1.0 破題：「金剛」的三層義——經名即教法

金剛（vajra），傳統譯為金剛石、鑽石，取其「世界上最堅硬的物質」之義。由此衍生的標準解讀是：般若如金剛，**無堅不摧**——能切斷一切煩惱、一切執著。

這個解讀有一個隱含的預設：金剛是一把**武器**。般若是刀，煩惱是被切的對象。

但如果般若是空性的智慧，而空性意味著無自性——那麼般若本身也不可能是一個「堅硬的實體」。**一把由空性構成的刀，怎麼可能是「世界上最堅硬的」？**

本文提出：「金剛」至少包含三層義理，逐層遞進，而每一層都預告了金剛經的核心教法。

#### 第一層：無處可砍（空）

金剛的意義不是「無堅不摧」——不是它比什麼都硬，能摧毀一切。金剛的意義是：**沒有一個東西可以讓你摧毀**。

不可壞（indestructible），不是因為它擋得住你的攻擊，而是因為**根本沒有目標**。你拿什麼武器都沒用——不是因為它比你的武器更強，而是因為它不在那裡讓你打。所以金剛可以是最硬的，也可以是最柔的——它不被任何一個「相」固定住。

般若不是一把鋒利的刀。般若是**無處可砍**。

這預告了即非結構的核心機制：「即非 X」不是用更大的力量摧毀 X，而是照見 X 本來就不是你以為的那個實體。沒有東西被摧毀——因為從來就沒有一個實體在那裡。

#### 第二層：清澈如水晶（照）

金剛不只是空的——它是**透明的**。

如同水晶球：本體永遠中性、清澈，但如實呈現背後的一切。水晶球後面是紅色，它顯示紅色；後面是藍色，它顯示藍色。水晶球本體從未改變，但善業惡業、一切因緣果報，透過它如實顯現。

楞嚴經中佛陀即以琉璃為喻說明此義：

> 「譬如琉璃，內懸明月，身心快然，妙圓平等，獲大安隱。一切如來密圓淨妙皆現其中。」[^1]

[^1]: 《大佛頂首楞嚴經》卷八，T19, No. 945。

琉璃本體透明——這是金剛的「空」（第一層）。明月懸於其中而照見——這是金剛的「照」（第二層）。一切如來密圓淨妙**皆現其中**——這是金剛的「不壞」（第三層）。

楞嚴經全經十卷共出現十三處琉璃比喻，[^2] 皆指向同一教法：心性本自清淨透明如琉璃，能映現一切而不被染污。

[^2]: 另見卷十「十方世界及與身心如吠瑠璃，內外明徹」（T19, No. 945）；卷十「圓明精心於中發化，如淨瑠璃內含寶月」（同上）。

金剛般若的第二層義理即在於此：般若不只是「空」——般若同時是**照見**。它不加不減、不善不惡、不判斷不篩選，只是如實映現。這就是為什麼金剛經的功能是鏡子（見 S5-P02）：鏡子本體透明中性，但你站到它面前，你的一切如實顯現。

此層直接橋接 S6 楞嚴經系列——楞嚴的「照」，正是金剛般若的水晶球功能在修行診斷上的展開。

#### 第三層：不生不滅（金剛不壞）

金剛看起來像水晶般柔透——但它是最堅硬的。

然而這個「堅硬」不是物質意義上的堅硬。金剛之所以不可壞，是因為它**不生不滅**——它從來就沒有被生出來，所以不可能被消滅。你不能摧毀一個從未誕生的東西。

這回扣 S1-P02 的法身論：法身超越三世，不在時間中生滅。金剛般若就是法身的功能面——它不是修出來的，不是得到的，是**本來如是**。所以它不可壞——不是因為它很強，而是因為它的存在方式根本不在「壞/不壞」的範疇內。

#### 三層遞進

```
第一層：空 ——— 無處可砍（沒有目標）     → 即非結構
第二層：照 ——— 清澈如水晶（如實映現）   → 公案鏡
第三層：不壞 — 不生不滅（本來如是）       → 法身
```

三層不是並列的，是遞進的：你先看到「空」（第一層），然後發現空不是什麼都沒有——它在**照**（第二層），最後你發現這個照見的本體根本不生不滅——你連「它是空的」這個判斷都要放下（第三層）。

金剛經的經名，已經預告了整部經的教法。

## 三、核心論證

### 3.1 窺基的映射：「說→遍計，即非→依他，是名→圓成」

窺基（T1702）的三性映射：

```
「說 X」          → 遍計所執性：命名本身即是妄執，「說」就是分別
「即非 X」        → 依他起性：緣起無自性，破斥遍計後顯現依他
「是名 X」        → 圓成實性：離遍計之執，顯現真實
```

窺基的邏輯：凡夫聽到佛陀「說」莊嚴，立刻以為有一個實體叫做莊嚴——這就是遍計。「即非」破除這個遍計，顯示莊嚴是緣起的（依他起）。「是名」在依他起上照見圓成實。

### 3.2 本文的替代映射：「X→依他，即非→破遍計，是名→圓成」

本文提出的映射：

```
「X」            → 依他起性：緣起法的暫時施設，作為語言指涉的對象
「即非 X」        → 否定遍計所執性：X 不是你以為的那個實體
「是名 X」        → 圓成實性的安立：名言假立不礙使用，如實知其為名
```

### 3.3 兩種映射的結構性差異與論證

**差異核心**：X（被命名的對象）的存在論地位。

- **窺基**：「說 X」= 遍計。命名本身就是遍計——佛陀一開口「說」，聽者立刻生遍計。佛說的「X」是聽者心中的投射
- **本文**：「X」= 依他起。莊嚴佛土、般若波羅蜜、三十二相——這些是依他起的緣起法，不是完全虛無。遍計不在「X」本身，而在聽者**對 X 的實有化執著**

**本文映射的論證要點**：

1. **如果「說」= 遍計，佛陀的「說」也是遍計嗎？**
   - 窺基的映射有一個張力：佛陀也在「說」，但佛陀不應生遍計
   - 窺基可能的回應：「說」是指聽者聽到後生起的遍計，不是佛陀本身的遍計
   - 但這就意味著「說」的遍計不在語言本身，而在聽者——那麼「X」本身（作為佛陀所指的對象）仍然可以是依他起

2. **依他起不是遍計——這是唯識的根本教義**
   - 解深密經（T676）明確區分：遍計是「假名安立」的妄執，依他起是「緣生自性」的實在
   - 如果「說 X」= 遍計，那 X 就是完全虛無的——但金剛經不是在說莊嚴完全不存在
   - 金剛經是在說：莊嚴**存在**（依他起），但不是你以為的那種存在（遍計）

3. **「是名」的功能在兩種映射中不同**
   - 窺基：「是名」= 圓成實。但圓成實是離二執的真如，為什麼用「名」來表達？
   - 本文：「是名」= 在圓成實的認知中重新安立名言。你還是叫它「莊嚴」，但你知道這只是名言假立——名言不礙圓成，圓成不廢名言

4. **兩種映射可能是互補的，不是矛盾的**
   - 窺基的映射從**聽者**的角度出發：你聽到「說」就生遍計，「即非」破你的遍計，「是名」讓你見到真實
   - 本文的映射從**對象**的角度出發：X 是依他起的緣起法，「即非」破你對 X 的遍計執，「是名」在 X 的依他起上照見圓成
   - 兩者的「是名 = 圓成」一致，差異在於遍計的定位
   - **指月的學術貢獻**：不是推翻窺基，而是**補充窺基**——從另一個角度照見同一個結構

> **[前瞻 S5-P02]**：為什麼同一個即非結構會產生兩種映射？因為它們針對不同境界的人說話。窺基的映射是給「用想的人」（概念分析），指月的鏡子映射是給「超越想但還未證的人」（直接照見）。兩者看似不同，其實是同一面鏡子在不同距離照出的影像——禪宗三關（初關/重關/牢關）的框架在 P02 展開。

### 3.5 即非結構在前半 vs 後半的功能差異

- **前半（§1–16，破我執）**：即非結構主要用於切除「我」相關的執著——四相、福德、佛身
- **後半（§17–32，破法執）**：即非結構升級——切除的對象包括般若本身、佛法本身、教法本身
- **遞進邏輯**：從破「我對法的執著」到破「法本身的實有性」——唯識二障（煩惱障/所知障）的精確對應

## 四、跨學科會通

### 4.1 語言哲學：Performative Contradiction 的正面使用

⚠️ 啟發性類比

- 維根斯坦的梯子比喻（Tractatus 6.54）：「必須超越這些命題，然後才能正確地看世界」——與金剛經的筏喻驚人相似
- Austin 的 performative utterance：「即非 X，是名 X」不是在描述空性，而是在**執行**空性——這句話的說出本身就在做它所說的事
- Derrida 的 sous rature（under erasure）：寫下一個詞然後劃掉——「即非」就是語言層面的 sous rature

### 4.2 形式邏輯：否定之否定 ≠ 黑格爾辯證法

⚠️ 啟發性類比

- 表面上，「X → 即非 X → 是名 X」像黑格爾的正反合（thesis-antithesis-synthesis）
- 但關鍵差異：黑格爾的合題是更高層次的綜合，佛教的「是名 X」不是綜合——它是**回到原處，但認知方式改變了**
- 圓成實不是遍計和依他的綜合——它是在依他上遠離遍計後的如實照見

## 五、中道校正

### 5.1 ❌ 三性映射不是嚴格等價

本文提出的「即非 = 三性」映射是一個**詮釋性框架**，不是宣稱金剛經的作者（或佛陀）在使用唯識術語。

- 金剛經成立（約 2–3 世紀）早於唯識三性理論的系統化（4–5 世紀）
- 歷史先後不影響義理分析的有效性——但必須區分「原意」和「後世詮釋」
- 本文的立場：三性框架是**照見**即非結構的有效工具，不是宣稱即非結構「就是」三性

### 5.2 ❌ 不可用三性框架馴化金剛經

三性映射的最大危險：把金剛經的野性收編進一個整齊的系統。

- 如果讀者讀完本文後覺得「啊，原來即非就是三性，我懂了」——那本文就失敗了
- 金剛經拒絕被任何框架完全捕獲——包括唯識框架
- 本文提供的三性映射是一根手指——它指向月亮，但它不是月亮（指月之旨）
- 更深層的教學效果，見姊妹篇 S5-P02

### 5.3 ❌ 黑格爾類比的嚴格限制

- 「正反合」類比僅在形式結構上有啟發性——即非結構不是辯證法
- 佛教的「否定」不產生更高的綜合——它照見本來面目
- 此類比在論文中必須標記為 ⚠️ 並明確其邊界

2. 本文提出替代映射（X→依他起，即非→破遍計，是名→圓成），與窺基的映射（說→遍計，即非→依他，是名→圓成）形成互補
3. 兩種映射的差異在於遍計的定位——窺基從聽者角度，本文從對象角度——共同揭示即非結構的完整義理機制
4. 無著/世親已分析即非結構但未用三性術語，窺基完成了這一步——學術進程清晰可考
5. 三性映射是有效的詮釋工具，但不能也不應馴化金剛經——鏡子的教學效果，見 S5-P02
