# Series 5：切 — 金剛般若波羅蜜經

**Vajracchedikā — The Diamond That Cuts**

Papers #01–10

---

## 系列簡介

S3 花了整個系列建立唯識工具箱（百法明門論 100 個法的精密分類）。S5 的工作不是推翻 S3，而是升維——工具箱還在，但你不再把工具當成實體。金剛經貫穿全經的「即非 X，是名 X」公式，不是修辭，而是唯識三性的壓縮表達。
