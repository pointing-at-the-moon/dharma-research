## 金剛經宏觀結構

```
前半（§1–16）：約眾生空——破四相（我、人、眾生、壽者）
    ║
    §17 樞紐：須菩提重問同一問題，佛陀給出更深答案
    ║
後半（§17–32）：約法空——連教法、功德、佛性都不可執
```

### 核心結構特徵

1. **公案體**：「即非 X，是名 X」遞歸否定結構不是邏輯論證，是一面鏡子——照見你能不能看懂
2. **自我重演**：§17 使整部經摺疊為雙層結構——同一問題，兩層深度
3. **功德較量遞增**：§4 → §8 → §11–§15 → §24 → §32，賭注持續升級，答案永遠一樣
4. **自毀裝置**：經文用功德作誘餌，最終指向功德本身的空性

## 學術獨創定位

**命題一**：金剛經的「即非 X，是名 X」三段式 = 唯識三性（trisvabhāva）的壓縮表達。

- 「X」= 依他起性（paratantra-svabhāva）——緣起法的暫時施設
- 「即非 X」= 否定遍計所執性（parikalpita-svabhāva）——切除對 X 的實有執著
- 「是名 X」= 圓成實性（pariniṣpanna-svabhāva）——照見 X 如實相，名言假立不礙使用

**命題二**：「金剛」包含三層遞進義理——經名即教法。

| 層 | 金剛的意義 | 佛法對應 | 系列連結 |
|---|-----------|---------|---------|
| **一、無處可砍** | 沒有目標讓你摧毀 | 空性——般若無自性 | S5 即非結構 |
| **二、清澈如水晶** | 本體中性透明，如實呈現一切 | 照見——楞嚴經水晶球喻 | S5-P02 公案鏡 → S6 楞嚴經 |
| **三、不生不滅** | 從未誕生，不可能被消滅 | 法身不壞——本來如是 | S1-P02 法身超三世 |

兩個命題共同構成 S5 的學術貢獻：經名本身即是教法（命題二），經文結構即是教法（命題一）。

## 教學危險與中道校正重點

| 危險 | 對治 | 出現在 |
|------|------|--------|
| **斷滅見**（nihilism）：「一切都空，什麼都不用做了」 | 依他起如幻有 ≠ 無；「空」是遍計空，不是依他空 | P09（主戰場）、P04、P06 |
| **口頭般若**：概念上理解「空」但行為不變 | 知/證區分——知道空不等於證到空。公案鏡照見你懂還是不懂 | P02 |
| **功德執著**：把功德較量當成修行 KPI | 功德較量是教學裝置，不是計分板 | P05 |
| **否定修行**：「法尚應捨」所以不用修了 | 筏喻——先過河才捨筏 | P04 |
| **假裝無執**：自稱無所住但實際上在執「無所住」 | 這是最隱蔽的法執，金剛經後半專門對治 | P07 |
