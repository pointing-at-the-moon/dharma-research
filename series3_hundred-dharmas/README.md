## 系列簡介

三部典籍交織：《百法明門論》= 體（分類系統），《八識規矩頌》= 用（操作手冊），《解深密經》= 教（經典授權）。100 法 + 12 頌全部涵蓋。

從般若進入唯識：認識你的心的每一個零件，看見它們如何運作，然後轉識成智。
