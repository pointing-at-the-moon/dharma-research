## 系列簡介

以天親菩薩《大乘百法明門論》為核心，將心識活動解構為五位百法的精密架構。百法不是名相記誦的清單，而是修行地圖——五位排列本身就是「從識到真如」的路徑；三能變揭示八識的動態結構；三性觀法提供觀照的方法論透鏡。

S1 準備法器，S2 以般若照見空性，S3 以唯識展開有的精密結構——空有互補，缺一不可。S5 再將 S3 的工具箱從「實有」升維為「如幻施設」。
