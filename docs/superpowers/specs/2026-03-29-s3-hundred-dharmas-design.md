# S3 — 行：百法明門 · 解深密經 · 八識規矩頌

> Design Spec · 2026-03-29
> Series 3 of the 指月 dharma-research project

---

## Context

S1（信）建立發心基礎與唯識基本概念，S2（解）以心經為綱展開般若空性。S3（行）的任務是給讀者**完整的唯識觀察工具箱**——讀完 S3，讀者應該能用百法分類精確描述任何一個心理現象。

S2 說「五蘊皆空」，S3 說「好，讓我告訴你五蘊裡面到底裝了什麼」。這是從般若（看穿）到百法（拆解）的自然遞進。

## 論文結構（13 篇）

### 區塊一：總綱 + 八識（P01–P06）

#### S3-P01 一切法無我：百法總綱

- **核心任務**：給讀者整個 S3 的地圖
- 百法明門論原文（722 字）全文呈現並逐句解析
- 五位百法分類架構：心法 8 → 心所有法 51 → 色法 11 → 不相應行法 24 → 無為法 6
- **解深密經**：引入三性作為百法分類的認識論基礎
- S2 銜接：S2 說「五蘊皆空」，S3 說「那讓我告訴你五蘊裡面到底裝了什麼」
- 研究資產：`research/hundred-dharmas/百法明門論_學術版_完整.md`

#### S3-P02 三能變：八識全景

- **核心任務**：一張圖看完八識的結構關係
- 八識規矩頌的第一頌（體性頌）× 4 組 = 4 段速寫
- 三能變框架：初能變（阿賴耶）→ 二能變（末那）→ 三能變（前六識）
- **不深入**任何一識——只建立全景
- 研究資產：`research/hundred-dharmas/心法/綜合研究/八識義理研究_學術版.md`

#### S3-P03 阿賴耶識：生命的硬碟

- **核心任務**：最深的識，最不可觀察的識
- 八識規矩頌三頌全展開（體性 + 功能 + 轉依）
- 三藏義：能藏、所藏、我愛執藏
- 種子六義、薰習、瀑流比喻
- **解深密經**：阿陀那識偈——「阿陀那識甚深細，一切種子如瀑流，我於凡愚不開演，恐彼分別執為我」
- 轉依目標：大圓鏡智
- 研究資產：`research/hundred-dharmas/心法/阿賴耶識/`

#### S3-P04 末那識：看不見的我執

- **核心任務**：為什麼你無法停止「我」？
- 八識規矩頌三頌全展開
- 恒審思量 + 四煩惱常俱（我癡、我見、我慢、我愛）
- 末那和阿賴耶的鏡像關係（末那執阿賴耶為「我」）
- 轉依目標：平等性智
- 研究資產：`research/hundred-dharmas/心法/末那識/`

#### S3-P05 第六意識：唯一能照鏡子的識

- **核心任務**：唯一能修行的識，也是唯一能造重業的識
- 八識規矩頌三頌全展開
- 三類運作：五俱意識、獨頭意識（獨散、夢中、定中）
- 51 心所全部可以與第六識相應——這是它強大的原因
- **核心論述——照鏡子**：第六意識是八識中唯一具備自我覺察能力的識。末那識恆常執我卻不知道自己在執，阿賴耶識儲藏一切卻沒有自覺，前五識接收感官卻不知「是我在接收」。唯有第六意識能反觀自身的心理活動——「我在想」「我在生氣」「我在執著」。這個照鏡子的能力，就是一切修行的入口。
- 修行的樞紐：第六識是唯一能主動轉向的識（回扣 S1-P06 覺察即轉向）
- 轉依目標：妙觀察智
- 研究資產：`research/hundred-dharmas/心法/意識/`

#### S3-P06 前五識：感官的直接性

- **核心任務**：為什麼眼見不一定為實？
- 八識規矩頌三頌全展開（前五識共用一組頌）
- 前五識的特徵：現量、性境、不帶名言
- 與第六識的合作：前五識取境 → 第六識分別 → 末那執我
- 轉依目標：成所作智
- 研究資產：`research/hundred-dharmas/心法/眼識/` 等五個

### 區塊二：心所有法（P07–P11）

#### S3-P07 遍行與別境：心理引擎

- **核心任務**：每一個心念都有這 10 個零件在運作
- **遍行 5**（觸、作意、受、想、思）：八識全有，無一例外
  - 作意回扣 S1-P06（覺察即轉向）——S1 講的修行樞紐，這裡給出唯識精確定位
  - 受、想呼應 S2 五蘊（受蘊 = 遍行之受，想蘊 = 遍行之想）
  - 思 = 行蘊核心 = 造業的真正主體
- **別境 5**（欲、勝解、念、定、慧）：需要特定條件才生起
  - 定 + 慧 = 止觀的心所基礎
- 研究資產：`research/hundred-dharmas/心所有法/遍行/` + `別境/`

#### S3-P08 善心所：修行的燃料

- **核心任務**：「善」不是道德判斷，是心所的化學成分
- **善 11**：信、精進、慚、愧、無貪、無瞋、無癡、輕安、不放逸、行捨、不害
- 三善根（無貪、無瞋、無癡）= 對治三毒的直接藥方
- 慚與愧的區別：對自己的標準 vs 對他人的覺知
- 信 = S1 整個系列的心所基礎——S1「信」在百法中的精確定位
- 研究資產：`research/hundred-dharmas/心所有法/善/`

#### S3-P09 根本煩惱與不定：輪迴的引擎

- **核心任務**：6 個根本煩惱是一切痛苦的源頭，4 個不定心所則視因緣而定
- **根本煩惱 6**：貪、瞋、癡、慢、疑、惡見
  - 五鈍使（貪瞋癡慢疑）+ 五利使（惡見分五：身見、邊見、邪見、見取見、戒禁取見）
  - 與末那識四煩惱的關係：末那的我癡我見我慢我愛是根本煩惱的「常駐版」
- **不定 4**（惡作、睡眠、尋、伺）：不善不惡，視因緣而定
  - 放在根本煩惱之後形成對比——煩惱是確定惡的，不定是看因緣的
- 研究資產：`research/hundred-dharmas/心所有法/根本煩惱/` + `不定/`

#### S3-P10 隨煩惱：煩惱的枝末

- **核心任務**：20 種隨煩惱，全部依根本煩惱而起
- **小隨 10**（忿、恨、覆、惱、嫉、慳、誑、諂、害、憍）：各自獨立生起，情境性強
- **中隨 2**（無慚、無愧）：一切不善心都有——只要起不善，必定無慚無愧
- **大隨 8**（掉舉、昏沉、不信、懈怠、放逸、失念、散亂、不正知）：一切染污心都有
- 統一論述邏輯：從小到中到大，染污範圍逐層擴大
- 研究資產：`research/hundred-dharmas/心所有法/隨煩惱/`

### 區塊三：色法 + 不相應 + 無為 + 收束（P11–P13）

#### S3-P11 色法與不相應行法：外面的世界

- **核心任務**：你以為有「外面」，唯識說這都是識的投影
- **色法 11**：五根 + 五境 + 法處所攝色
  - 唯識無境：不是否認桌子存在，是說你認知到的桌子是識的活動
  - S2「色即是空」的百法版：色法 = 依他起性中的一部分
- **不相應行法 24**：得、命根、眾同分、異生性、無想定、滅盡定、無想報、名身句身文身、生住異滅、流轉定異相應勢速次第、時方數、和合不和合
  - 這 24 個「法」是概念建構——不是心，不是心所，不是物質，但認知框架離不開它們
  - 時間、空間、數量——全是不相應行法，是識的活動的「標籤」
- 合併理由：各自單獨撐不起完整論文，合在一起形成「唯識怎麼看外部世界」的完整論述
- 研究資產：`research/hundred-dharmas/色法/` + `不相應行法/`

#### S3-P12 無為法：百法的終點

- **核心任務**：100 個法裡面，只有 6 個不是造作出來的
- **無為法 6**：虛空無為、擇滅無為、非擇滅無為、不動滅無為、想受滅無為、真如無為
- 真如無為 = 圓成實性 = 一切法的真實面目
- **解深密經**：真如與三性的最終收束
- 篇幅較短但義理深度高——從有為到無為的轉折
- 研究資產：`research/hundred-dharmas/無為法/`

#### S3-P13 轉識成智：回家

- **核心任務**：百法不是拆給你看的，是拆了才能轉
- 八識規矩頌轉智偈：四智圓明
  - 前五識 → 成所作智
  - 第六識 → 妙觀察智
  - 末那識 → 平等性智
  - 阿賴耶識 → 大圓鏡智
- **解深密經**：三轉法輪——為什麼唯識是「第三轉」、為什麼需要前兩轉作基礎
- S3 終極回扣：P01 說「一切法無我」，P13 說「無我之後，智慧自然現前」
- 銜接 S4：工具箱完整了，接下來用 AI 現象挑戰這個框架的邊界
