# Series 8：顯 — 大方廣佛華嚴經
**Manifest (顯) — The Buddhāvataṃsaka-mahāvaipulya-sūtra: The Full Display of the Dharmadhātu**

S1–S3 建構唯識地圖，S4 以質疑拆解盲信，S5（金剛經）切一切執著，S6（楞嚴經）以五十陰魔照妖鏡驗證真偽，S7（法華經）以一佛乘回收一切法門。S8（華嚴經）的工作不是再建立什麼，而是**顯**——被前七個系列逐步建立、切割、驗證、回收的那個對象，在華嚴中以法界自身的密度全相呈現。沒有 S5 的切與 S6 的照，S8 的顯就變成執著；有了 S7 的攝，S8 的顯才有容器承接。

本系列以四條方法論宣言貫穿：

1. **阿含滲漏法（leakage method）**——不再使用 S1–S7 的「阿含種子→大乘開展」模型，而是反向論證：阿含中佛陀明說不答的形上問題、明說不可思議的對象、與天人的對話、梵天勸請的敘事，都是「同一個無形廣播在人類頻道上的滲漏點」。
2. **十玄門作為寫作姿態**——P01–P10 每篇以一扇玄門承擔「以特定方式失敗以指向不可說對象」的工作，形式即內容。
3. **止語點（failure point）制度**——每篇必標「語言到此為止」的位置，指月在 S8 是字面實踐。
4. **經先論後在結構層的徹底化**——P01–P10 對應普賢十大願王（佛親說次第），而非十玄門（祖師論述次第）。

核心錨點：法藏《修華嚴奧旨妄盡還源觀》「真如本覺也，妄盡心澄，萬象齊現」。
